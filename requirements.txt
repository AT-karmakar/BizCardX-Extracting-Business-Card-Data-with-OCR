This code implements a Streamlit web application called "BizCardX." It allows users to upload images of business cards, extracts relevant information using the easyOCR library, displays the extracted data, and provides options to upload this data to a PostgreSQL database. It offers functionalities to view, modify, and delete the uploaded business card information.
required packages are:
easyOCR, Pandas, Psycopg2, matplotlib, streamlit
